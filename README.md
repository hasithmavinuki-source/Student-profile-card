# Student Profile Card

This is a simple **Student Profile Card** project created using **HTML and CSS**.  

## Features
- Displays student **name**, **stream**, and a short **bio**.
- Includes a **profile photo placeholder**.
- Stylish design with:
  - Rounded corners
  - Shadow effects
  - Gradient background
  - Modern fonts

## How to View
1. Open `index.html` in a web browser.
2. You will see the student card with styling applied.

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Poppins)
